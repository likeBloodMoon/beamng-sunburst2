# beamng-sunburst2
A tuning-focused addon for the Sunburst2 2.5L F4 that turns it into a proper experimental monster. This pack adds a fully tunable Lua-powered ECU, a variable-geometry turbo kit and a 15-speed manual setup built to handle stupid amounts of torque.

🧠 Deluxe Lua ECU

Adjustable rev limiter & cut time

Custom idle RPM

Power Multiplier slider (0.2×–5×) that scales the whole torque curve

Toggleable “indestructible engine” mode

Fine-tune damage thresholds (cylinder walls, head gasket, piston rings, rods, torque limits)

🌀 Stage 2 Variable Geometry Turbo Kit

Low/high boost multipliers with smooth transition RPM

Adjustable spool time for how “lazy” or “snappy” the turbo feels

Exposes VGT data (boost, vane position, sound level) to electrics for custom gauges/HTML UI

⚙️ 15-Speed Manual Transmission & Transaxle

Close lower gears for acceleration, long upper gears for top speed/cruise

Beefed-up torque capacity to survive high power + crazy power multiplier setups
